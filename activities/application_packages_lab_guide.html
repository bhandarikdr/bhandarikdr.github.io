Q. No. 4. Advanced Report Formatting (Long Document Management)

Task: Create a formal 5-page document consisting of three chapters following strict corporate formatting styles:

Part A: Style Definitions

  • Normal Style: Arial font, size 11, 1.15 line spacing, 0.6cm left indent.
  • Heading 1: Times New Roman, size 14, Bold, Level 1 numbering (1, 2, 3...). Spacing: 12mm before, 6mm after.
  • Heading 2: Arial font, size 12, Bold, Level 2 numbering (1.1, 1.2...). Spacing: 6mm before, 6mm after.
  • Heading 3: Arial font, size 11 (Regular / Not Bold), Level 3 numbering (1.1.1...). Spacing: 6mm before, 3mm after.

Part B: Document Structure & Nesting Rules

  • Page 1-2: Chapter 1 • Page 3-4: Chapter 2 • Page 5: Chapter 3
  • Nesting Rule 1: Under every Heading 1, you must include at least two Heading 2s.
  • Nesting Rule 2: Under every Heading 2, you must include at least three Heading 3s.

Part C: Layout, Headers & Pagination

  • Headers & Section Breaks: Use Next Page Section Breaks so each chapter displays its own distinct title/chapter number in the header.
  • Tables: Insert at least one fully styled grid table on each of the 5 pages.
  • Front Matter: Prepend an unnumbered/separate section featuring an automated Table of Contents (TOC) followed by a Table of Tables (TOT).
  • Page Numbering: Front matter pages must use lowercase Roman numerals (i, ii, iii) in the footer, while chapters must restart at standard Arabic numbers (1, 2, 3...).

Part 4 Databases Concept
Database Application Task (MS Access)

Objective: Design a relational database named Einstein_Academy.accdb to manage student and faculty information.

  1. Table Definitions: Create Student Details table with fields: StudentID (PK), FirstName, LastName, DateOfBirth, ContactNumber, EmailAddress.

  3. Forms Design: Create an input form to log 5-10 student records.
  4. Queries: Build filters to fetch records (e.g., Marks > 70 or LastName = "Rai").
  5. Report Generation: Use Report Wizard to create printable layouts.
