🧠 Overview Computer fundamentals

Computer – an electronic device that processes data and instructions. It follows the IPO cycle: Input → Process → Output.

Data = raw facts (e.g., student marks). Information = processed, organized data (e.g., sorted list with totals).

⚡ CPU (Central Processing Unit) – brain of computer. Contains:

  • ALU – arithmetic & logic operations
  • Control Unit – coordinates components
  • Registers – ultra-fast storage

🔹 Input devices: keyboard, mouse, scanner, microphone … 🔸 Output devices: monitor, printer, speaker, projector.

Software = programs + data. System software (OS) vs. Application software (Word, Excel).

📝 Assessment prep Key points to remember
  • 🔹 IPO cycle – Input, Process, Output.
  • 🔹 Five generations: technology, speed, language, memory, I/O devices.
  • 🔹 Storage – bit (0/1) → byte → KB (1024 bytes) → MB → GB → TB → PB → EB → ZB → YB.
  • 🔹 Classifications: micro (PC, laptop), mini, mainframe, super.
  • 🔹 Mobile computing – wireless communication, hardware (smartphones, tablets) and software (OS, apps).
  • 🔹 Advantages: productivity, portability, cloud sync. Disadvantages: battery, security, interference.
