Lab Activity: Computer Organization & I/O

Comprehensive Laboratory Work Guide

⚙️ Setup Before you begin

🎯 Objectives What you will learn
  • Differentiate between Computer Architecture and Computer Organization.
  • Identify and classify input, output, and memory devices.
  • Explain the working of primary and secondary memory (RAM, ROM, Cache, Hard Disk, Optical, Flash).
  • Describe microprocessor architecture and its key features.
  • Understand bus architecture and communication ports.
🧠 Overview Computer Organization Fundamentals

Computer Architecture defines WHAT the computer does (the design, the instruction set). Computer Organization defines HOW it does it (the hardware implementation).

🧩 Key Components:

  • Input Unit – keyboard, mouse, scanner, microphone, etc.
  • Memory Unit – Primary (RAM, ROM, Cache) & Secondary (HDD, SSD, CD, DVD, Pen Drive)
  • CPU – ALU, Control Unit, Registers
  • Output Unit – Monitor, Printer, Speaker, Plotter

⚡ Microprocessor – the brain on a chip. Follows Fetch → Decode → Execute cycle. RISC (simpler instructions, faster) vs CISC (complex instructions, fewer lines of code).

🔌 Bus Architecture – Data bus, Address bus, Control bus. Ports – Serial, Parallel, USB, SCSI, IEEE 1394, PCI.

🧪 Challenge Quiz yourself
1. Which memory is directly accessible to the CPU?
  • a) Secondary memory
  • b) Primary memory
  • c) Cache memory
  • d) Both b and c
🔍 Show answer
✅ d) Both Primary memory (RAM/ROM) and Cache are directly accessible to the CPU.
2. Which bus carries memory addresses from CPU to other components?
  • a) Data bus
  • b) Control bus
  • c) Address bus
  • d) Internal bus
🔍 Show answer
✅ c) Address bus carries memory addresses. It is uni-directional from CPU.
3. Which of the following is a Non-impact printer?
  • a) Dot Matrix printer
  • b) Daisy wheel printer
  • c) Laser printer
  • d) Line printer
🔍 Show answer
✅ c) Laser printer is a non-impact printer. Others are impact printers.
4. What does RISC stand for?
  • a) Reduced Instruction Set Computer
  • b) Random Instruction Set Computer
  • c) Regular Instruction Set Computer
  • d) Reliable Instruction Set Computer
🔍 Show answer
✅ a) Reduced Instruction Set Computer – designed for simpler, faster instructions.
📝 Assessment prep Key points to remember
  • 🔹 Architecture (WHAT) vs Organization (HOW)
  • 🔹 Input devices: Keyboard, Mouse, Scanner, MICR, OMR, BCR, etc.
  • 🔹 Output devices: Monitor (CRT, LCD, LED, Plasma), Printer (Impact, Non-impact), Plotter, Speaker
  • 🔹 Memory: Primary (RAM – S-RAM, D-RAM; ROM – PROM, EPROM, EEPROM; Cache), Secondary (Magnetic, Optical, Flash)
  • 🔹 Microprocessor: RISC vs CISC, Coprocessor, DSP, Transputer
  • 🔹 Bus: Data (bi-directional), Address (uni-directional), Control (uni-directional)
  • 🔹 Ports: Serial, Parallel, USB, SCSI, IEEE 1394, PCI
